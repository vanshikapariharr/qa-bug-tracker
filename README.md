# 🐞 QA Bug Tracker

A multi-page bug tracking application that allows users to create, manage, and track bugs through a complete lifecycle. Built with a focus on quality assurance practices and end-to-end testing.

---

## 🌐 Live Demo

👉 https://bug-tracker-qa.netlify.app/bugs

---

## 🚀 Features

- Full CRUD functionality for bug management  
- Status lifecycle: Open → In Progress → Resolved  
- Form validation and error handling  
- Filter bugs by status  
- Persistent data using localStorage  
- Global state management with React Context API  

---

## 🧪 Testing (Cypress)

- 15+ end-to-end test cases  
- Covers:
  - Navigation flows  
  - Form submission  
  - Validation errors  
  - Filtering functionality  
  - Status updates
